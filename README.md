# LAB 2 - Mobexler

TP Sécurité Android réalisé avec la machine virtuelle **Mobexler**.

## Environnement utilisé
- Machine virtuelle : Mobexler (VMware)
- Émulateur Android : Android Studio AVD (Pixel 5, Android 16)
- Connexion ADB via port forwarding VMnet8

## Travail réalisé
- Configuration et démarrage de Mobexler
- Connexion ADB entre l'émulateur Android et la VM Mobexler
- Réalisation des exercices de pentest Android
